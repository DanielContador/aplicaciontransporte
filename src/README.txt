Estsa aplicación fue desarrollada utilizando ionic con angular.

para poder correr la aplicación es necesario instalar las siguientes librerias:

 npm i -g @angular/cli


ng add @angular/material

npm install @ionic/storage-angular

npm install mapbox-gl --save


   npm install cordova-sqlite-storage 
   npm install @awesome-cordova-plugins/sqlite 

si da error:
npm install @awesome-cordova-plugins/core --save


$ npm install cordova-plugin-geolocation
$ npm install @awesome-cordova-plugins/geolocation
$ ionic cap sync


es necesario crear un nuevo proyecto con ionic start y luego reemplazar la carpeta src del mismo con esta...
después de eso instalar las librerias.
